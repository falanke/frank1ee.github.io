Hexo 博客从 Gitcafe 迁移到 Coding

Gitcafe 在16年3月2日宣布已加入 Coding，5月31日后将会停止所有服务并删除所有的账户资料和项目，
既然如此那就将博客迁移至 Coding.net 吧。

根据 gitcafe 的提示页面一键迁移即可，迁移完成后可以在 Coding 看到相对应的 repo，复制 HTTPS 链接
在 Hexo 的 config 文件中修改 deploy，直接覆盖 gitcafe 的链接：

# Deployment
  ## Docs: https://hexo.io/docs/deployment.html
   deploy:
     type: git
     repo:
          github: https://github.com/userName/userName.github.io.git
          gitcafe: https://git.coding.net/userName/repoName.git

hexo g hexo d成功后到 Coding 的项目页面中进入「演示」页面：

「检测应用」后强制开启，然后点击开启功能后进入功能页面，显示高级选项后「运行环境」选择 HTML，「访问域名」填上自己心仪的自定义域名确认后就可以点击「一键部署」了，随后看到部署成功就可以通过该域名访问了。
